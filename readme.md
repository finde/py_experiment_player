# Building App
python setup.py build

# Running App
Open Stress_and_Addiction.exe
Go to browser and open 127.0.0.1:8000

# Open Debug Mode
Press Ctrl+D

# Development
/server is the python runner for local webServer
/src is the UI Interface for rendering template

## Server Development
run `./dev.sh`

## UI Development
run `yarn run dev`
